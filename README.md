# vue-todolist

A very simple todolist made with Vue.js.

The todolist utilizes local storage to store the todolist items.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
